# io_evolutions

Projekt jest implementacją EMAS (Evolutionary Multiagent Systems) w środowisku 
[Watchmaker](http://http://www.watchmaker.uncommons.org/). Został wykonany w ramach przedmiotu Inteligencja Obliczeniowa 
na AGH.
